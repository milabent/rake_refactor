# RakeRefactor

Refactor your Ruby projects

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rake_refactor', git: 'https://github.com/milabent/rake_refactor.git'
```

And then execute:

    $ bundle

## Usage

    $ rake refactor:rename[OldName,NewName]
