# RakeRefactor

Refactor your Ruby projects

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rake_refactor'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rake_refactor

## Usage

    $ rake refactor:rename[OldName,NewName]
